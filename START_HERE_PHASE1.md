## 🆘 Если что-то не работает

### Ошибка при миграции БД
**Проблема:** SQL выдает ошибку
**Решение:** 
- Убедитесь, что вы в правильном проекте Supabase
- Проверьте, что скопировали весь SQL код
- Попробуйте выполнить по частям

### Ошибка при импорте товаров
**Проблема:** Товары не загружаются
**Решение:**
- Проверьте формат файла (.xlsx)
- Убедитесь, что SKU и название заполнены
- Создайте категории перед импортом

### Ошибка при деплое на Vercel
**Проблема:** Деплой не проходит
**Решение:**
- Проверьте переменные окружения
- Убедитесь, что код загружен на GitHub
- Проверьте логи в Vercel Dashboard

### Не работает аутентификация
**Проблема:** Не могу войти после деплоя
**Решение:**
- Проверьте настройки URL в Supabase Auth
- Убедитесь, что Redirect URLs добавлены
- Очистите кэш браузера

## 🛠️ Технологический стек

### Frontend
- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **UI Library:** React 19
- **Styling:** Tailwind CSS 4
- **Components:** Shadcn/ui
- **Icons:** Lucide React

### Backend
- **Database:** PostgreSQL (Supabase)
- **Auth:** Supabase Auth
- **API:** Supabase Client
- **Security:** Row Level Security (RLS)

### Deployment
- **Platform:** Vercel (рекомендуется)
- **Alternative:** Docker + VPS

## 📊 База данных

### Таблицы (10 шт)
1. `organizations` - Организации
2. `profiles` - Профили пользователей
3. `organization_users` - Связь пользователей с организациями
4. `categories` - Категории товаров
5. `locations` - Склады и магазины
6. `suppliers` - Поставщики
7. `products` - Каталог товаров
8. `inventory` - Остатки по локациям
9. `transactions` - Все операции
10. `transaction_items` - Позиции транзакций

### Безопасность
- ✅ Row Level Security (RLS) на всех таблицах
- ✅ Мультитенантность (изоляция данных по организациям)
- ✅ Политики доступа для каждой таблицы
- ✅ Автоматические триггеры для updated_at

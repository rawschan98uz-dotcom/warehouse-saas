## 🚀 Как запустить (5 минут)

### Шаг 1: Настройте Supabase (2 минуты)

1. Создайте проект на [supabase.com](https://supabase.com)
2. В SQL Editor выполните скрипт из `supabase/schema.sql`
3. Скопируйте URL и API ключ из Settings → API

### Шаг 2: Настройте переменные окружения (30 секунд)

Откройте `.env.local` и замените:

```env
NEXT_PUBLIC_SUPABASE_URL=https://ваш-проект.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=ваш_anon_ключ
```

### Шаг 3: Установите зависимости (1 минута)

```bash
npm install
```

### Шаг 4: Запустите проект (30 секунд)

```bash
npm run dev
```

Откройте http://localhost:3000

### Шаг 5: Зарегистрируйтесь (1 минута)

1. Перейдите на `/register`
2. Создайте аккаунт
3. Войдите в систему
4. Готово! 🎊

## ⚠️ Важно: Ошибка сборки

При запуске `npm run build` вы увидите ошибку:
```
Error: Invalid supabaseUrl: Must be a valid HTTP or HTTPS URL.
```

**Это нормально!** Ошибка возникает потому, что в `.env.local` указаны placeholder значения.

**Решение:**
1. Настройте Supabase (см. SUPABASE_SETUP.md)
2. Замените URL и ключ в `.env.local`
3. Запустите `npm run build` снова

После настройки Supabase проект соберется без ошибок.

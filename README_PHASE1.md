# 📦 Warehouse Management System - Фаза 1

Современная SaaS система для управления складом, товарами и остатками с поддержкой мультитенантности.

## 🎯 Текущая версия: Фаза 1

### ✨ Новые возможности Фазы 1:

- ✅ **Улучшенная категоризация товаров** - иерархия категорий, расширенные атрибуты (бренд, цвет, размер, штрих-код)
- ✅ **Массовая загрузка товаров** - импорт через Excel с автоматической валидацией
- ✅ **Модуль инвентаризации** - проверка остатков с ручным вводом и автоматическим расчетом расхождений
- ✅ **Улучшенные перемещения** - перевод товаров между складом и магазином

## 📋 Основные возможности

### Управление товарами
- Полный CRUD для каталога товаров
- Категоризация с иерархией (категории → подкатегории)
- Расширенные атрибуты: бренд, цвет, размер, штрих-код
- Массовая загрузка через Excel
- Автоматическая генерация SKU
- Поиск и фильтрация

### Управление складом
- Учет остатков по локациям (склад + магазин)
- Инвентаризация с ручным вводом
- Автоматический расчет расхождений
- Перемещения между локациями
- Приход и списание товаров
- Уведомления о низких остатках

### Транзакции
- Приход товаров
- Продажи
- Перемещения между локациями
- Списание товаров
- История всех операций

### Отчеты и аналитика
- Дашборд с ключевыми метриками
- Графики продаж (30 дней)
- Топ товаров
- Статистика по категориям
- Экспорт в CSV

### Управление
- Мультитенантность (организации)
- Роли пользователей (admin, manager, staff)
- Управление локациями
- База поставщиков
- Настройки валюты (UZS/USD)

## 🗄️ База данных

### Основные таблицы

- `organizations` - Организации
- `profiles` - Профили пользователей
- `organization_users` - Связь пользователей с организациями
- `categories` - Категории товаров (с иерархией)
- `products` - Каталог товаров (с атрибутами)
- `locations` - Склады и магазины
- `inventory` - Остатки товаров
- `transactions` - Все операции
- `transaction_items` - Позиции транзакций
- `suppliers` - Поставщики

### Новые таблицы Фазы 1

- `product_imports` - История импорта товаров
- `inventory_sessions` - Сессии инвентаризации
- `inventory_items` - Позиции инвентаризации

## 🛠️ Технологии

- **Frontend:** Next.js 16.2.2 (App Router), React 19, TypeScript
- **UI:** Shadcn/ui + Tailwind CSS 4
- **Backend:** Supabase (PostgreSQL + Auth)
- **Deployment:** Vercel
- **Excel:** xlsx library
